# دليل الاختبار السريع - نظام إدارة العيادة

## بيانات اختبار النظام

### المستخدمون الاختباريون

استخدم هذه البيانات لتسجيل الدخول والاختبار:

```
1. مدير النظام:
   - البريد: admin@clinic.com
   - الكلمة: admin123
   - الصلاحيات: كاملة

2. طبيب:
   - البريد: doctor@clinic.com
   - الكلمة: doctor123
   - الصلاحيات: إدارة المرضى، المواعيد، الفواتير، التقارير

3. موظف استقبال:
   - البريد: receptionist@clinic.com
   - الكلمة: reception123
   - الصلاحيات: إدارة المرضى والمواعيد فقط
```

## خطوات الاختبار

### 1. اختبار تسجيل الدخول والصلاحيات

**كمدير نظام:**
- [ ] سجل الدخول ببيانات المدير
- [ ] تحقق من ظهور تبويب "المستخدمون"
- [ ] تحقق من ظهور تبويب "التقارير"

**كطبيب:**
- [ ] سجل الدخول ببيانات الطبيب
- [ ] تأكد من عدم ظهور تبويب "المستخدمون"
- [ ] تحقق من ظهور تبويب "التقارير"

**كموظف استقبال:**
- [ ] سجل الدخول ببيانات موظف الاستقبال
- [ ] تأكد من عدم ظهور تبويب "المستخدمون"
- [ ] تأكد من عدم ظهور تبويب "التقارير"

### 2. اختبار إدارة المستخدمين (كمدير فقط)

- [ ] انقر على تبويب "المستخدمون"
- [ ] أضف مستخدم جديد:
  - اسم: "أحمد علي"
  - بريد: ahmed@clinic.com
  - دور: طبيب
- [ ] ابحث عن المستخدم الجديد
- [ ] عدّل بيانات المستخدم
- [ ] حذف المستخدم

### 3. اختبار إدارة المرضى

**كمدير أو طبيب:**
- [ ] انقر على تبويب "المرضى"
- [ ] أضف مريض جديد:
  - الاسم: "محمد أحمد"
  - البريد: patient@email.com
  - الهاتف: 0501234567
  - العمر: 35
  - السجل الطبي: "سكري النوع الثاني"
- [ ] ابحث عن المريض الجديد
- [ ] عدّل بيانات المريض
- [ ] تحقق من الرسائل (نجاح/خطأ)

### 4. اختبار المواعيد

- [ ] انقر على تبويب "المواعيد"
- [ ] أضف موعد جديد:
  - اختر مريض موجود
  - حدد التاريخ والوقت
  - أضف ملاحظات
- [ ] غيّر حالة الموعد
- [ ] احذف الموعد

### 5. اختبار الفواتير

- [ ] انقر على تبويب "الفواتير"
- [ ] أضف فاتورة جديدة:
  - اختر مريض
  - حدد المبلغ
  - اختر الحالة
- [ ] عدّل الفاتورة
- [ ] غيّر الحالة من معلقة إلى مدفوعة

### 6. اختبار التقارير

**كمدير أو طبيب:**
- [ ] انقر على تبويب "التقارير"
- [ ] تحقق من عرض الإحصائيات:
  - إجمالي المرضى
  - إجمالي المواعيد
  - المواعيد المكتملة
  - الإيرادات الإجمالية
- [ ] حدد فترة زمنية مختلفة
- [ ] تحميل التقرير (CSV)

### 7. اختبار الصلاحيات المتقدمة

**اختبار حذف البيانات:**
- [ ] كموظف استقبال: حاول حذف مريض → يجب أن تظهر رسالة خطأ
- [ ] كطبيب: حاول حذف مريض → يجب أن ينجح الحذف

## قائمة التحقق الشاملة

- [ ] تسجيل الدخول يعمل بشكل صحيح
- [ ] الصلاحيات تطبق بشكل صحيح
- [ ] إضافة البيانات تعمل بشكل صحيح
- [ ] التعديل يحفظ البيانات بشكل صحيح
- [ ] البحث يعمل بشكل صحيح
- [ ] الحذف يعمل بشكل صحيح
- [ ] الرسائل (خطأ/نجاح) تظهر
- [ ] التقارير تعرض البيانات الصحيحة
- [ ] تحميل التقرير يعمل
- [ ] الواجهة سهلة الاستخدام وبالعربية

## نصائح

- استخدم أدوات المطور (F12) لرؤية الأخطاء
- جرب بمتصفحات مختلفة
- اختبر مع كميات كبيرة من البيانات
- تحقق من الاتصال بـ Supabase قبل الاختبار
- استخدم بيانات واقعية قدر الإمكان

## مشاكل شائعة وحلولها

### المشكلة: لا يمكن إضافة مريض
**الحل:**
1. تحقق من الاتصال بـ Supabase
2. تأكد من ملء جميع الحقول المطلوبة
3. تحقق من صيغة البريد الإلكتروني

### المشكلة: الصلاحيات لا تطبق بشكل صحيح
**الحل:**
1. تحقق من دور المستخدم في قاعدة البيانات
2. أعد تحميل الصفحة (F5)
3. سجل الخروج وسجل الدخول مرة أخرى

### المشكلة: التقارير لا تظهر
**الحل:**
1. أضف بعض البيانات أولاً
2. تأكد من اختيار الفترة الزمنية الصحيحة
3. تحقق من الأخطاء في وحدة التحكم (Console)
